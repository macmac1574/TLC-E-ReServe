Drop your real assets here:

  - qr-code.svg (or .png)   → Real QR code linking to your APK download URL
  - app-mockup.svg (or .png) → Screenshot of the app's home screen (no device frame)
  - screenshot-1.svg ... 6  → Real app screenshots (9:16 ratio recommended)
  - app.apk                 → Your built Android APK package

The current SVG files in this folder are placeholders generated to make the
landing page render out-of-the-box. Replace them with real assets before
shipping.

If you change file extensions (e.g. .svg → .png), update the matching src=""
attributes in ../index.html.
